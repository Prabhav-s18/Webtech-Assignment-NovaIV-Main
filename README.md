# NovaIV 

NovaIV Cameras is a multi-page website designed to help people who apply for the company. The site showcases the products , job opportunities, and allows users to apply for roles through a validated application form.

## Our Main Pages
### Home Page (index.html)
- Introduction to NovaIV Cameras and its mission
- Product showcase (cameras, lenses, accessories)
- Search bar interface
- Shipping information table
- Acknowledgement of Country section

### Jobs Page (jobs.html)
- Multiple job listings (Frontend, Backend, Marketing, Camera Specialist)
- Detailed responsibilities and requirements
- Sidebar with company benefits
- “Apply” buttons linking to application form

### Application Page (apply.html)
- Fully functional job application form
- HTML5 validation using:
- Regex patterns (e.g., postcode, phone, job reference)
- Required fields
- User-friendly layout with grouped inputs
- Skill selection via checkboxes

### About Page (about.html)
- Team member profiles with roles and contributions
- Class schedule section
- Team information table
- Group photo and additional details

## How to Run
- Download or clone the repository
- Open the project folder
- Open index.html in any web browser
